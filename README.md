## MyBlog

blog 是使用 Node.js + MongoDB + 其它客户端框架开发的个人博客系统,前后端分离,仿简书模板.
服务端有: express 版 , koa 版, egg.js
客户端有: react 版 , vue 版， vue3.0， nuxt.js
移动端有: react native 版, ionic2.0 版

UI: iview

ps: 博客就是用来检验所学知识的产物

一些灵感： animation gameful 音乐播放 视屏播放
